# algopack-api
node module for building and hosting api for Algopack :Online version

[NPM](https://www.npmjs.com/package/algopack)
## Installation
```javascript
npm install algopack -g
```
OR
```javascript
sudo npm install algopack -g
```
 ## usage
 ```javascript
algopack build
 ```

OR
 ```javascript
 sudo algopack build
 ```
 - api will be created in _api folder
